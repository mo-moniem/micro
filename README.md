
## About Project

This Project has two microservice auth and note crud operation
in communicate with each other and use message broker Rabbitmq

#### auth start point 

- cd auth-user
- php artisan ser
- php artisan migrate --seed
- php artisan test 

#### note start point

- cd note-service
- php artisan ser
- php artisan migrate --seed
- php artisan test
